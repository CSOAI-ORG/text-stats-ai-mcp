# text-stats-ai-mcp

MCP server for text analysis tools.

## Tools

- **word_count** — Comprehensive text statistics (words, sentences, paragraphs)
- **reading_time** — Estimate reading time with readability scores
- **keyword_density** — Analyze keyword frequency and density
- **sentiment_score** — Lexicon-based sentiment analysis

## Usage

```bash
pip install mcp
python server.py
```

## Rate Limits

50 calls/day per tool (free tier).
